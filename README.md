# 🏡 TripNest – Airbnb Clone

A full-stack Airbnb-style web application where users can explore homes, view details, add favourites, and hosts can manage property listings.

---

## 🚀 Features

### 👤 User Side

* 🏠 Browse all available homes
* 🔍 View detailed home information
* ❤️ Add / Remove favourites
* 📖 View favourite listings
* 📅 Booking page UI

### 🧑‍💼 Host Side

* ➕ Add new homes
* ✏️ Edit existing listings
* ❌ Delete homes
* 📋 View all hosted homes

---

## 🛠️ Tech Stack

* **Frontend:** EJS, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Templating Engine:** EJS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/rimisoni22-ux/TripNest.git
cd TripNest
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the server

```bash
node app.js
```

---

## 🌐 Environment Variables

Create a `.env` file (optional) and add:

```
MONGO_URI=your_mongodb_connection_string
PORT=3000
```
